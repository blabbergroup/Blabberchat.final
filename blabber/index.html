This website started out as a project for a class at Mississippi State University. The directive was to create something that is different from what is currently available on the market.

The group decided to create a chat application. We took the best parts of other applications and combined them to create something new and innovative.

Blabber was the result.

So what is Blabber?

Blabber is a chat app that allows users to customize their server. It has the ability to add modules to the server based on what you need. It allows you to change colors of text and backgrounds of different parts of the server. It adds voice or video if you need it.(Functionality coming later)

By default it will have file-sharing, GIF search, emojis, and rooms. (Other things will be added but this should be more than enough to start with)
